2. ¿Qué es un sitio web?

Qué es una página web

Una página web es un documento digital al que se puede acceder mediante Internet utilizando un navegador. Puede contener texto, imágenes, videos, enlaces y elementos interactivos. Las páginas web están creadas principalmente con lenguajes como HTML, CSS y JavaScript, y sirven para informar, comunicar, entretener o permitir realizar acciones como comprar o estudiar.

Tipos de páginas web

Existen diferentes tipos de páginas web según su función. Algunas son informativas, como blogs o periódicos digitales; otras son tiendas online donde se pueden comprar productos; también existen redes sociales para interactuar con otras personas y plataformas educativas para aprender en línea. Además, las páginas pueden ser estáticas, cuando siempre muestran el mismo contenido, o dinámicas, cuando cambian según el usuario o la interacción.

Navegadores y servidores

Un navegador es el programa que usamos para entrar a páginas web y visualizar su contenido. Algunos ejemplos son Google Chrome, Mozilla Firefox y Safari. El servidor, en cambio, es una computadora especializada que almacena las páginas web y las envía cuando un usuario las solicita. El navegador y el servidor trabajan juntos para que podamos acceder a la información en Internet.

Cómo funciona internet

Internet es una gran red mundial de computadoras y dispositivos conectados entre sí. Cuando una persona entra a una página web, su dispositivo se conecta a través de un proveedor de Internet y envía una solicitud al servidor donde está almacenada la página. Para que la comunicación funcione se utilizan direcciones IP, sistemas DNS y protocolos como HTTP y HTTPS, que permiten localizar y transferir información entre dispositivos.

3. ¿Cómo viaja la información por Internet?

  • El usuario escribe una dirección web en el navegador.
  • El navegador envía una solicitud al servidor correspondiente.
  • La información se divide en pequeños paquetes de datos.
  • Los paquetes viajan por diferentes rutas de Internet mediante cables, routers y antenas.
  • El servidor responde enviando los datos solicitados.
  • El dispositivo del usuario recibe y reorganiza los paquetes.
  • Finalmente, el navegador muestra la página web completa en la pantalla.

    • 4. Conceptos básicos de HTML y CSS

    HTML se utiliza para estructurar el contenido de una página web, mientras que CSS sirve para diseñar y dar estilo a los elementos visuales.

    HTML

    <html>: código que se utiliza para estructurar y desplegar una página web y sus contenidos.
<head>: provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.
<body>: representa el contenido de un documento HTML. Solo puede haber un elemento como este en un documento.
<h1>: representa un título en el documento.
<p>: representa un texto en el documento.
<img>: representa una imagen en el documento.
<a>: crea un hipervínculo para dirigirse a otros sitios, como web, archivos, partes de la misma web, etc.
<ul> y <li>: sirven para crerar listas numeradas o no numeradas.

    CSS

    CSS permite cambiar colores, tamaños, márgenes, animaciones y el diseño completo de una web.

    MD

    Un archivo .md es un documento de texto simple que no contiene ningún otro elemento. En él, pueden in­tro­du­ci­r­se símbolos en el texto para definir el formato de ciertas secciones.
